# formazione_sou

Repository dedicata al percorso di Academy DevOps.

Raccoglie tutto il materiale prodotto durante il corso: esercizi, script, appunti e risorse di studio.

## Contenuto

Il materiale è organizzato per argomento e cresce man mano che il percorso avanza.

## Contribuire

Ogni nuovo esercizio o risorsa va inserito nella cartella di riferimento, con un nome file chiaro e descrittivo.
